Dashboards

This page is set up for dashboard case studies, reporting workflows, and interactive projects built to make data easier to explore and explain.

What This Page Is Ready For

You can use this layout for Tableau, Power BI, Shiny, JavaScript dashboards, or static reporting pages. Each project card below already has slots for the problem, audience, tools, screenshot, code notes, and a live link.

  • Add a public Tableau link or embed once a dashboard is published.
  • Swap placeholder summaries with the real project question, workflow, and insights.
  • Attach R, SQL, or Python code notes to the projects where they matter most.

Platforms You Can Feature

  • Tableau Public: Public dashboards, story views, and interactive filters.
  • Power BI: Published reports, screenshots, and write-ups of the data model.
  • R Shiny: Interactive apps tied to research or statistical workflows.
  • Custom Web Dashboards: HTML, CSS, and JavaScript builds hosted like the rest of this site.
